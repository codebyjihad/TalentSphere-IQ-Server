# Coding Interview Platform

A full-featured coding interview platform with real-time video, code editor, chat, and automated feedback.

---

## Features

- VSCode-Powered Code Editor
- Authentication with Clerk
- 1-on-1 Video Interview Rooms
- Dashboard with Live Statistics
- Microphone & Camera Toggle, Screen Sharing and Recording
- Real-time Chat Messaging
- Secure Code Execution in Isolated Environment
- Automatic Feedback (Success / Fail based on test cases)
- Success Animation and Failure Notifications
- Practice Problems Page (Solo Coding Mode)
- Room Locking (Maximum 2 Participants)
- Background Jobs using Inngest (Asynchronous Tasks)
- REST API with Node.js and Express
- Data Fetching and Caching using TanStack Query
- PR Analysis and Code Optimization with CodeRabbit
- Git & GitHub Workflow (Branches, Pull Requests, Merges)
- Deployment on Sevalla (Free-tier Friendly)
- Environment Variable Configuration
